NUTRICALC Web v1.0 FINAL
Sistem Pendukung Asuhan Gizi Terintegrasi

Cara pakai:
1. Ekstrak file ZIP ini.
2. Buka file index.html menggunakan Google Chrome.
3. Gunakan tab: Antropometri, Kebutuhan Gizi, Asupan, Cairan, % Pemenuhan, Diabetes & Insulin.
4. Data berjalan lokal/offline.

Catatan final v1.0:
- Tanpa dashboard.
- Tanpa CPPT.
- Watermark AF.
- Interpretasi IMT dewasa:
  <18,5 = Kurus
  18,5–25,0 = Normal
  >25,0–27,0 = Overweight
  >27,0 = Obesitas
- BBI Brocca Modifikasi Asia = (TB - 100) × 0,9.
- Recall URT memiliki 4 pilihan dropdown dengan total otomatis.
- Asupan dilengkapi satuan: energi kkal, protein gr, lemak gr, KH gr.

- index.html

- app.js
